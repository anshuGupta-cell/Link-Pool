## Overview
Link Pool is a web application that allows users to organize and share multiple links through a single profile page. It works like a personal link hub where all social, portfolio, or important URLs can be accessed from one place.

---

## Features
- Create personal link profile
- Add and manage multiple links
- Single shareable page
- Clean and responsive UI
- Fast deployment using Vercel

---

## Tech Stack
- Frontend: React.js
- Styling: CSS / Tailwind (if used)
- Deployment: Vercel
